CREATE a new user /users POST A JSON object holding a name.
Example:
{
 "name": "John Doe"
} 		A JSON object with all object data of new student along with new id in uuid.v4 form. Example:
{
 "name": "John Doe"
 "id": "a31e0082-f522-45f2-b703-8d64fab0fb4b"
}
UPDATE the username of a user /users/[id] PUT A JSON object with updated name.
Example:
{
 "name": "Jane Doe"
} 		A JSON object of all data for the updated user.
Example:
{
 "id": 1,
 "name": "Jane Doe",
 "favoriteMovies": []
}
CREATE a new entry from a user's favorite movies /users/[id]/[movie_title] POST None Returns a message with the added movie title and the user id.
DELETE an entry from a user's favorite movies /users/[id]/[movie_title] DELETE None Returns a message with the deleted movie title and the user id.
DELETE existing user /users/id DELETE None Returns a message informing that the user has been removed
